# Dotfiles

My personal configuration files for setting up my work environment in a way I like.

## Installation

```sh
git clone git://github.com/jimf/dotfiles ~/.dotfiles
cd ~/.dotfiles
./install
```

## Environment

I use the same configuration in both Mac OS X and Ubuntu Linux.
